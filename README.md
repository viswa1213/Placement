## ğŸš€ How to Run

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- Any IDE (VS Code, IntelliJ IDEA, Eclipse) or command line

### Running a Java Program

1. **Clone the repository**
   ```bash
   git clone https://github.com/viswa1213/Placement.git
   cd Placement
   ```

2. **Navigate to the desired directory**
   ```bash
   cd java/Day1
   ```

3. **Compile and run**
   ```bash
   javac GCD.java
   java GCD
   ```
